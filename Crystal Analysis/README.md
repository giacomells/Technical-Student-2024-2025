These are some examples of EverestCrystal class. 
This class is inherited from the baseCollimator class.

The examples in the Channeling directory are made for analysing the efficiency of the channeling mode of the crystals for different lengths of the crystals, bending angles of the crystals and impacting angles of the pencil beam.

To change the angle in the horizontal plane(s,x) of a crystal you need to change the variable 'tilt'.
