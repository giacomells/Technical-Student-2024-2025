# Crystal Analysis

This directory contains notebooks focused on crystal behavior and examples based on the `EverestCrystal` class.

## Contents

- `CombinedCharacterisationOfTheCrystal.ipynb`
- `GaussianBeamCrystalExample.ipynb`
- `exampleRematchingTunes.ipynb`
- `Channeling Efficiencies studies/` for efficiency scans versus crystal length, bending angle, tilt, and impacting angle.

## Notes

- These notebooks are analysis/example notebooks rather than shared library code.
- `exampleRematchingTunes.ipynb` depends on an external optics JSON sequence from outside this folder.
- To change the horizontal plane angle of a crystal, modify the `tilt` variable in the relevant notebook.
